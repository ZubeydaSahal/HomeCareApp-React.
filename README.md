# HomeCareApp – React Frontend

HomeCareApp er en frontend-applikasjon utviklet i React for administrasjon og oversikt i et hjemmetjenestesystem. Applikasjonen er designet for å gi helsepersonell og administratorer en enkel og effektiv brukeropplevelse.

## Teknologi
- React
- JavaScript (evt. TypeScript)
- CSS
- REST API
- Git

## Funksjonalitet
- Autentisering og rollebasert tilgang
- Oversikt over brukere (pasienter / ansatte)
- Henter data fra backend via REST API
- Komponentbasert struktur og gjenbruk


##  Prosjektstruktur
```md
src/
components/
pages/
services/
models/
App.js
```
